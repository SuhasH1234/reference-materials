> css selector: used to select the html elements which we want to style
> 5 categories: Simple sel
		Combination sel
		Pseudo class sel
		Pseudo elements sel
		Attribute sel
> simple sel: name, id, class
> combination sel: it selects elements based on some relationship between them
> Pseudo class sel: selects elements based on certain specific state
> Pseudo elements sel: selects and styles a part of an element
> Attribute sel: selects elements based on attribute or attribute values

> element selector: selects html elements based on element name
< id selector: uses the id attribute to an html element to specify the certain element
> class selector: selects html elements with a specific class attribute
> universal selector: selects all html elements in page

> types of css: inline, internal, external
> inline css: used to apply a style for a single element
> internal css: used to style one single html page which has unique style and is declared in head section with <style>
